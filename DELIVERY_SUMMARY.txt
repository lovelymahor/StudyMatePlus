🚀 GETTING STARTED
═══════════════════════════════════════════════════════════════════════════════

  2. Update server/.env with your MONGO_URI
  3. Start backend:   cd server && npm run dev
  4. Start frontend:  cd client && npm start

  6. Test: Register → Login → Profile
